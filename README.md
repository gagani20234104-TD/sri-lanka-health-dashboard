# 🌿 Sri Lanka Health & Air Quality Dashboard

An interactive Streamlit dashboard analysing WHO health and air quality indicators for Sri Lanka.

**Module:** 5DATA004C – Data Science Project Lifecycle  
**University:** University of Westminster  
**Dataset:** [Air Quality Indicators for Sri Lanka – WHO via HDX](https://data.humdata.org/dataset/who-data-for-lka)

## 📊 Features

- Interactive trend line chart over selected year range
- Bar chart of average values per year
- Distribution histogram of indicator values
- Top 10 indicators by latest value (horizontal bar chart)
- Sex disaggregation pie chart
- Raw data table with CSV download
- Sidebar filters: indicator selector, year range slider, sex multi-select

## 🛠 Installation & Running Locally

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/sri-lanka-health-dashboard.git
cd sri-lanka-health-dashboard

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

## 📦 Dataset

- **Source:** WHO via Humanitarian Data Exchange (HDX)
- **URL:** https://data.humdata.org/dataset/who-data-for-lka
- **Coverage:** Multiple health indicators for Sri Lanka across several years
- **Rows:** 100+ observations
- **Key indicators:** Air pollution attributable deaths, tuberculosis, malaria, health system metrics
