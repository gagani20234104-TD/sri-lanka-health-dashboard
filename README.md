# Sri Lanka Health and Air Quality Dashboard

An interactive Streamlit dashboard analysing WHO health indicators for Sri Lanka.

Module: 5DATA004C - Data Science Project Lifecycle
University: University of Westminster
Dataset: Air Quality Indicators for Sri Lanka - WHO via HDX
Link: https://data.humdata.org/dataset/who-data-for-lka

## Live App

https://sri-lanka-health-dashboard-mnhaj4qdwfdo9w2n8bjln6.streamlit.app/

## Features

- Interactive trend line chart showing health indicator changes over time
- Bar chart showing average values per year
- Histogram showing distribution of values
- Pie chart showing breakdown by sex
- Sidebar filters for indicator selection, year range and sex
- Raw data table with CSV download option

## Dataset

Source: World Health Organisation via Humanitarian Data Exchange
URL: https://data.humdata.org/dataset/who-data-for-lka

app.py is the main Streamlit application
